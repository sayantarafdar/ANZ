## ANZ Bank Analytics Challenge

Project: Developed a machine learning solution for ANZ Bank to predict loan default risk using customer demographic and financial data, enabling improved profitability, reduced risk, and data-driven loan approval decisions. 

### Responsibilities:

Conducted Exploratory Data Analysis (EDA) to uncover trends in income, credit score, and loan behavior. <br>
Cleaned and preprocessed data by handling missing values and encoding categorical variables. <br>
Built Logistic Regression and Random Forest models to analyze loan default patterns. <br>
Improved prediction performance through model tuning and feature selection. <br>
Evaluated models using accuracy, confusion matrix, precision, and recall. <br>
Generated Actual vs Predicted analysis tables to validate model accuracy and support business decision-making. <br>
Stored processed datasets and model results in Microsoft Fabric Data Lakehouse using Spark and Delta tables for scalable data management. <br>
Delivered actionable insights to minimize financial risk and improve loan approval decisions. <br>
Collaborated in an Agile environment with daily stand-ups and task tracking using Jira.
